## merchant 2.0
> 销售机器人


## 创建步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# 部署至开发环境
npm run javadev

# 部署至测试环境
npm run test

# 部署至预发布环境
npm run pre

# build for production with minification
npm run build


```

## 目录说明
``` bash

├── build                ## webpack构建相关
├── dist                ## 打包目录
├── src                 ## 开发目录
│   ├── views       ## 业务目录
│   │   └── Main  ## 首页
│   ├── router   ## 路由控制
│   ├── service  ##  服务层，主要管理API相关
│   └── store  ## vuex
│   └── libs  ## 其他功用组件
|   │   └── fetch  ## 请求接口页面



```
